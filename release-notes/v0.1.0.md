# v0.1.0

**Release Date:** 2026-02-04

This is the initial public release of tinyfly, a lightweight, API-driven animation engine.

## Highlights

- Complete animation engine with timeline-based playback
- Visual editor built with SolidJS
- Embeddable player for websites and apps
- JSON-first animation format for portability

---

## Added

### Core Engine

- **Timeline System** - Orchestrate multiple animation tracks with precise timing
- **Track-based Animation** - Animate any property with keyframe interpolation
- **Easing Functions** - Built-in easing (linear, ease-in, ease-out, cubic-bezier)
- **Clock Abstraction** - RAF-based clock for browsers, manual clock for testing
- **Playback Controls** - Play, pause, stop, seek, reverse, and speed control
- **Looping** - Finite loops, infinite loops, and ping-pong (alternate) mode

### Interpolation

- **Number Interpolation** - Smooth transitions between numeric values
- **Color Interpolation** - Animate colors (hex, rgb)
- **Array Interpolation** - Multi-value property animation

### Adapters

- **DOM Adapter** - Animate HTML elements with CSS transforms and styles
- **Canvas Adapter** - Render animations to HTML5 Canvas
- **SVG Adapter** - Animate SVG elements and attributes

### Motion Path

- **SVG Path Animation** - Animate elements along SVG paths
- **Auto-rotation** - Automatic rotation following path direction

### Serialization

- **JSON Export/Import** - Full timeline serialization to JSON
- **Deterministic Playback** - Same input always produces same output

### Export Formats

- **CSS Animation Export** - Generate CSS @keyframes
- **Lottie Export** - Convert to Lottie JSON format
- **GIF Export** - Render animation as animated GIF

### Editor

- **Visual Timeline** - Drag-and-drop keyframe editing
- **Live Preview** - Real-time animation preview with DOM/Canvas toggle
- **Property Panel** - Edit keyframe values and easing curves
- **Curve Editor** - Visual cubic-bezier curve editor
- **Element Panel** - Manage scene elements (rect, circle, text, image)
- **Track Panel** - Add, remove, and organize animation tracks
- **Preset Library** - Pre-built animation presets (fade, slide, bounce, etc.)
- **Project Management** - Save/load projects with localStorage persistence
- **Undo/Redo** - Full history support with keyboard shortcuts
- **Keyboard Shortcuts** - Comprehensive shortcuts for common actions
- **Responsive Design** - Mobile-friendly editor with collapsible sidebars

### Player

- **Lightweight Embed** - Minimal player for embedding animations
- **Auto-registration** - Automatic target discovery via data attributes
- **Playback API** - Programmatic control (play, pause, seek, etc.)

### UI/UX

- **Splash Screen** - Animated landing page with tinyfly branding
- **Status Bar** - Version display and attribution footer
- **Onboarding** - First-time user guided tour
- **Animation Gallery** - Example animations showcase

## Technical Details

### Dependencies

- **Runtime:** Zero dependencies for core engine
- **Editor:** SolidJS, @solidjs/router
- **Build:** Vite, TypeScript, Vitest

### Browser Support

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Bundle Size

- Core engine: ~15KB (gzipped)
- Full editor: ~75KB (gzipped)

---

## Known Issues

- Motion path animation requires SVG path strings (no visual path editor yet)
- GIF export may be slow for long animations
